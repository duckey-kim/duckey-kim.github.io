# 코드로 배우는 스프링 웹 프로젝트

## p.56 @Setter(onMethod_ = @Autowired ) 에러

기존에는 jdk11 만 깔려있던 상태였습니다.
책을 따라서 설정을 하였는데 계속 에러가 나서 .m2의 repository를 여러번 지우고를 반복했습니다.
문제는 해결이 되지 않아서 jdk1.8.0 설치후 sts.ini의 설정을
```
-vm
C:\Program Files\Java\jdk1.8.0_261\bin\javaw.exe
```
으로 변경하였고 다시 .m2의 repository를 지우고 marven update를 하니 에러가 사라졌습니다.
